сломается сверка сумм.

## 2. Иерархия

Уровень территории в файле **не размечен колонкой** — он выводится только из
текста в колонке `A` и из порядка строк. Правило разбора:

| Уровень | Признак в тексте | Примеры |
|---|---|---|
| 0 — страна | `Республика Казахстан` | строка 7 |
| 1 — область | `Алматинская` (без слова «область») | строка 9 |
| 2 — район / горадминистрация | оканчивается на `район` или `г.а.` | `Балхашский район`, `Қонаев г.а.` |
| 3 — центр | префикс `город `, `г.`, `с.` | `г.Есик`, `с.Баканас`, `город Қонаев` |

Строка 8 пустая — она визуально отделяет республику от области.

Ловушки для парсера:

- Название области записано как **`Алматинская`**, без слова «область».
- `г.а.` = городская администрация. Для `Қонаев г.а.` и `Алатау г.а.` уровень 3
  записан по-разному: `город Қонаев` / `город Алатау` (слово целиком), тогда как
  у районов — сокращения `г.` / `с.`.
- Казахские названия идут в кириллице с казахскими буквами: `Қонаев` (буква `Қ`,
  U+049A), `Райымбекский`. Сравнение с русскими формами `Конаев`, `Раимбекский`
  требует нормализации.
- `с.Отеген Батыра` содержит пробел внутри названия — разделять по пробелу нельзя.

## 5. Замечания к содержанию

1. **Файл отражает деление ПОСЛЕ реформы 2022 г.** В нём нет ни Талдыкоргана, ни
   Аксуского, Алакольского, Ескельдинского, Каратальского, Кербулакского,
   Коксуского, Панфиловского, Сарканского районов — они переданы области Жетісу
   Указом Президента РК № 887 от 03.05.2022 (в силе с 08.06.2022). Это делает
   XLSX эталоном текущего деления при сверке с геоданными (см. `04-geodannye.md`).
   Состав — 9 районов и 2 города областного значения — совпадает с данными
   Бюро национальной статистики на 01.01.2026.
2. **Кегенский район присутствует отдельно от Райымбекского.** В старых
   справочниках (и в GADM) есть только Райымбекский. Разделение произведено
   Указом № 653 от 31.03.2018: у Райымбекского центр `с.Нарынкол`,
   у Кегенского — `с.Кеген`.
3. **Алатау г.а. — новая единица** (2024 г., из с. Жетыген Илийского района),
   в геоданных её нет ни в одном наборе.
4. **Қонаев г.а.** — бывший Капшагай (переименован Указом № 885 от 03.05.2022),
   ныне административный центр области; в GADM отсутствует полностью.
4а. **Названия центров даны в русской форме.** В официальных реестрах акиматов
   часть из них записана по-казахски: `с.Чунджа` → Шонжы,
   `с.Отеген Батыра` → Өтеген батыр. При стыковке с реестрами нужен словарь
   на обе формы.
5. **Илийский район показан как полностью сельский** (городское = 0), хотя его
   центр `с.Отеген Батыра` — крупный пригород Алматы (32 520 чел.). Это статус
   поселка, а не ошибка, но при картировании «городского» слоя даст пустоту
   вокруг Алматы. То же у Жамбылского (с.Узынагаш, 50 386 чел.) — по численности
   это город, по статусу село.
6. **Данные по городу Алматы и области Жетісу в файле отсутствуют.** Для
   сопоставимой картины по агломерации их нужно догружать отдельно.
7. Разрез — только пол и город/село. **Нет возрастных групп, нет динамики,
   нет национального состава.** Для риск-модели, чувствительной к возрастной
   структуре, этого файла недостаточно.

## 6. Рекомендации по загрузке

- Ключ единицы 2-го уровня — нормализованное название. Нормализация обязана
  приводить `Қ`→`К`, `Райымбекский`→`Раимбекский`, убирать суффиксы `район`/`г.а.`
- Хранить уровень (`country`/`oblast`/`district`/`center`) явной колонкой,
  восстановив его по правилу из раздела 2 — в исходнике его нет.
- `-` → `0`.
- Дату `2026-04-01` записать в метаданные набора: без неё цифры нельзя
  сопоставлять с бюджетными слоями других периодов.
