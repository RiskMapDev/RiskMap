# Публикация в интернете

Как получить ссылку, по которой систему открывает любой человек с любого
устройства. Всё бесплатно и без привязки карты.

> **Учётные записи на площадках создаёте вы.** Ассистент не регистрируется от
> вашего имени и не вводит пароли — это правило, а не техническое ограничение.
> Всё остальное подготовлено заранее.

---

## Что публикуется

**Только обезличенная база.** Файл `dist/riskmap-demo-obezlichennaya.dump`
(11 МБ). В нём:

| | Исходная база | Публикуемая |
|---|---|---|
| БИН и ИИН | настоящие | сгенерированные, с верным контрольным разрядом |
| Наименования ИП | «ИП ФАМИЛИЯ ИМЯ ОТЧЕСТВО» | «ИП Получатель 0001» |
| Наименования юрлиц | настоящие | **настоящие** — открытый реестр |
| Суммы, даты, территории | настоящие | **настоящие** |
| Оценки риска | настоящие | **настоящие** |
| Связи графа | 16 266 рёбер | 16 266 рёбер |

Обезличивается принадлежность, а не картина рисков: распределение уровней,
суммы и связи совпадают с исходными до последней цифры.

**Полный дамп в интернет не выкладывается никогда.** В нём БИН и ИИН 3 668
организаций и 3 413 получателей субсидий, из которых 3 183 — индивидуальные
предприниматели, то есть живые люди. Публикация такого рядом со словом
«критический риск» недопустима независимо от настроек доступа.

Пересоздать обезличенную базу:

```powershell
# 1. Копия рабочей базы — обезличивание необратимо
& "$env:USERPROFILE\pgsql\bin\pg_dump.exe" -U postgres -p 5433 -d riskmap `
    --format=custom -f copy.dump
& "$env:USERPROFILE\pgsql\bin\psql.exe" -U postgres -p 5433 -d postgres `
    -c "create database riskmap_demo;"
& "$env:USERPROFILE\pgsql\bin\pg_restore.exe" -U postgres -p 5433 `
    -d riskmap_demo --no-owner copy.dump

# 2. Обезличивание с проверкой
cd backend
$env:DATABASE_URL = "postgresql+psycopg://postgres@127.0.0.1:5433/riskmap_demo"
python -m scripts.anonymize --dry-run   # сначала посмотреть план
python -m scripts.anonymize             # заменить; при неудаче код возврата 1

# 3. Новые пароли — прежние совпадают с рабочей системой
python -m scripts.seed_access --reset-passwords
```

Скрипт заканчивает работу сквозной проверкой всех текстовых колонок схемы. Если
хоть один исходный идентификатор уцелел, он сообщает об этом и возвращает
ненулевой код. **Публиковать дамп, полученный с ненулевым кодом, нельзя.**

## Куда развернуть

Проверено 21 июля 2026. Тарифы меняются — сверьтесь перед началом.

| Слой | Площадка | Что даёт бесплатно |
|---|---|---|
| База | **Neon** | PostgreSQL с PostGIS, 0,5 ГБ, карта не нужна |
| API | **Vercel** (Python) | FastAPI как ASGI, Python 3.14 |
| Интерфейс | **Vercel** (Hobby) | Next.js 16 с SSR |

База 11 МБ против лимита 500 МБ — запас сорокакратный.

### ⚠️ Ограничение, о котором нужно знать заранее

**Vercel Hobby запрещает коммерческое использование.** В правилах прямо
сказано: тариф Hobby — только для личных некоммерческих проектов, а
коммерческим считается в том числе получение оплаты за создание, обновление
или хостинг сайта. Если карта рисков делается по заказу или за деньги, Hobby
формально не подходит, нужен Pro за 20 $ в месяц.

Бесплатная замена без этого ограничения:

| Слой | Площадка | Оговорка |
|---|---|---|
| API | **Render Free Web Service** | засыпает через 15 минут простоя, просыпается 30–60 секунд |
| Интерфейс | **Cloudflare Workers** + `@opennextjs/cloudflare` | бандл до 3 МиБ в сжатом виде |

База в обоих случаях — Neon.

### Отброшенные варианты

- **Render Postgres** — бесплатная база удаляется через 30 дней вместе с данными.
- **Supabase** — проект встаёт на паузу после 7 дней без запросов; для карты с
  редким трафиком это значит «не открывается, когда понадобилась».
- **Fly.io, Railway, Koyeb** — постоянного бесплатного тарифа больше нет.
- **PythonAnywhere** — не поддерживает ASGI, FastAPI не запустится.
- **CockroachDB** — нет PostGIS, а карта на нём и держится.

## Порядок действий

### 1. База на Neon

1. Зарегистрируйтесь на neon.com. **Выберите регион AWS**, не Azure: у
   Azure-регионов неактивные бесплатные проекты удаляются через 90 дней.
2. Создайте проект, скопируйте строку подключения.
3. Включите PostGIS и залейте дамп:

```powershell
$env:PGPASSWORD = "<пароль из строки подключения>"
& "$env:USERPROFILE\pgsql\bin\psql.exe" `
    "postgresql://<пользователь>@<хост>/<база>?sslmode=require" `
    -c "create extension if not exists postgis;"

& "$env:USERPROFILE\pgsql\bin\pg_restore.exe" `
    -d "postgresql://<пользователь>@<хост>/<база>?sslmode=require" `
    --no-owner --no-privileges dist\riskmap-demo-obezlichennaya.dump
```

### 2. API

Переменные окружения:

```
DATABASE_URL=postgresql+psycopg://<пользователь>:<пароль>@<хост>/<база>?sslmode=require
JWT_SECRET=<длинная случайная строка, не из этого файла>
CORS_ORIGINS=["https://<ваш-интерфейс>.vercel.app"]
```

`JWT_SECRET` обязательно новый. Значение по умолчанию годится для запуска на
своей машине и никуда больше.

### 3. Интерфейс

Одна переменная:

```
NEXT_PUBLIC_API_URL=https://<ваш-api>/api/v1
```

Она попадает в код страницы и видна всем — это адрес, а не секрет.

После первого развёртывания вернитесь к API и впишите в `CORS_ORIGINS`
настоящий адрес интерфейса. Пока он не совпадает, браузер будет отклонять
запросы, а на экране появится «Границы не загрузились».

## Учётные записи демонстрации

Пароли обезличенной базы отличаются от рабочих — их перевыдал
`seed_access --reset-passwords`.

**Публикуйте только `viewer`.** Он видит карту, списки и карточки, но ИИН ему
показывается замаскированным, а разделы импорта и администрирования закрыты.
Пароль `admin` держите при себе: под ним можно запустить импорт и изменить
настройки, то есть сломать демонстрацию.

Каждый вход и каждый просмотр полного идентификатора попадают в журнал
`audit_log` — он работает и в демонстрационном развёртывании.

---

## Что проверить после публикации

1. Вход под `viewer` — открывается панель.
2. Карта: слой «Государственные закупки», Балхашский район тёмно-красный,
   Кегенский серый.
3. Легенда называет долю показанных объектов.
4. Граф связей рисует узлы и рёбра.
5. Разделы «Импорт» и «Администрирование» под `viewer` недоступны.

Если карта серая целиком — не заполнен `NEXT_PUBLIC_API_URL` либо адрес
интерфейса не попал в `CORS_ORIGINS`.

## Чего это развёртывание не даёт

Бесплатные тарифы рассчитаны на демонстрацию, а не на работу.

- **Исходящий трафик Neon — 5 ГБ в месяц.** Геометрия территорий отдаётся из
  базы при каждом запросе карты. Несколько сотен открытий карты в месяц лимит
  выдержит, тысячи — нет; после исчерпания база останавливается до следующего
  месяца.
- **База засыпает** при простое. Первое открытие после паузы медленнее.
- **Загрузка файлов не переживёт перезапуск**: у бессерверного развёртывания
  нет постоянного диска. Мастер импорта в демонстрации показывать можно,
  полагаться на него — нет.

Для настоящей эксплуатации нужен сервер организации: там и персональные данные
остаются под контролем, и ограничений этих нет.
