# Источник границ

Файлы `kz_1.json` (области, ADM1) и `kz_2.json` (районы, ADM2) взяты из
открытого репозитория:

https://github.com/open-data-kazakhstan/geo-boundaries-kz (data/geojson)

Первоисточник — GADM 3.6 (https://gadm.org), лицензия Open Data Commons PDDL.

## Ограничения данных (важно)

- Границы **дореформенные** (до 2022 г.): 14 областей старого деления.
  Города республиканского значения (Астана, Алматы, Шымкент) отдельными
  регионами не выделены. «Старая» Алматинская область включает районы,
  позднее вошедшие в область Жетысу.
- Названия — латиница/английские. Русские имена и КАТО-коды добавляются
  из `territories/reference_data.py`.
- КАТО-кодов и населения в исходных данных нет.
  КАТО взяты с https://tenderplus.kz/kato (агрегатор; официальный
  первоисточник — stat.gov.kz, коды стоит перепроверить там).
  Население грузится позже через импорт-мастер.
- Рассинхрон: геометрия дореформенная, КАТО — постреформенные. Совпадение
  идёт по названию, поэтому у части объектов код отражает текущее деление,
  а полигон — старое. Два района Алматинской обл. (Алматы, Талдыкорган)
  оставлены без КАТО намеренно — однозначного соответствия нет.

При появлении официальных постреформенных границ РК с КАТО — заменить эти
файлы и перезапустить `python manage.py load_boundaries`.
