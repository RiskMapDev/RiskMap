ИАС «Интерактивная карта рисков» — запуск из архива
====================================================

Нужна одна программа: Docker Desktop.
    https://www.docker.com/products/docker-desktop/
Windows, macOS, Linux — подойдёт любой. Больше ничего ставить не надо.

Запуск
------

    Windows:     двойной щелчок по start.bat
    Mac/Linux:   ./start.sh

Первый запуск собирает контейнеры и занимает несколько минут — это
нормально. Скрипт сам создаст настройки, поднимет базу, зальёт в неё
данные из dist\ и откроет браузер.

    Интерфейс:  http://localhost:3000
    Вход:       admin / 123123123123
    Остановить: docker compose down

Другие учётные записи (пароль у всех тот же):
    analyst          — аналитик Алматинской области
    analyst.karasay  — аналитик только Карасайского района
    manager          — руководитель
    viewer           — просмотр

ВНИМАНИЕ: данные
----------------

В dist\ лежит дамп базы с ПЕРСОНАЛЬНЫМИ ДАННЫМИ: БИН и ИИН 3668
организаций и 3413 получателей субсидий, наименования и суммы выплат.
Не выкладывайте этот архив в публичный доступ и не передавайте тем,
кто не вправе обрабатывать такие сведения.

Пароль 123123123123 — демонстрационный. Для любой работы дальше
показа смените его:
    docker compose exec backend python -m scripts.seed_access --reset-passwords

Если что-то пошло не так
------------------------

    Логи базы:      docker compose logs db
    Логи бэкенда:   docker compose logs backend
    Живость API:    http://localhost:8100/ready
    Начать с нуля:  docker compose down -v  и снова start.bat

Запуск без Docker (Windows, вручную) — dist\КАК-РАЗВЕРНУТЬ.txt.
Полная документация — каталог docs\.
